ComputerInfo

In Development Preview Available Partially Open Source
ComputerInfo collects information about computers and sends it to a central location for storage and management. It can collect information about: model, serial numbers, memory, processors, graphics, printers, network cards, hard drives and more. It also contains tables for other devices so you can keep track of them as well. The tool is very useful for IT administrators who wants an overview of all devices within the corporation/institution. We’ve also measured a significant time reduction in the amount of time it takes to collect detailed information about all computers in an institution with about a hundred computers, we measured it to take around 2-3 hours compared to several days without the tool.

ClickThis

On Hold Preview Available
ClickThis is at the moment a prototype survey tool, that we started working on in mid 2011. It’s mainly centered around smartphones and tablets, but the plans is to create a desktop interface too. The survey designer is desktop only at the moment. There hasn’t been progress since June 2012, because we have been working on other projects and therefore ClickThis has been put on hold. ClickThis is planned to feature a full survey designer, where it will be possible to create questions, options, customize button colors, change button sizes, write descriptions, insert images, videos etc. The system uses social services for login, such as Facebook, Twitter, Github, Google and LinkedIn and more to come.
